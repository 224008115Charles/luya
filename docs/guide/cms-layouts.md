CMS Layouts
==========

Creating new CMS Layouts

JSON CONFIG
-----------

| Usabled object keys | Type   |  Description
| ------------------- | ------ | -------------
| placeholders		  | array  | creates new placeholders (with "var" and "label" keys)

```
{
    "placeholders" : [
    	{
    		"var" : "content", 
    		"label" : "Inhalt"
    	}
    ]
}
```
